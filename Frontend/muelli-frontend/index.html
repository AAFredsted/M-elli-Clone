About Mülli

Mülli is a project for the Master Practical Course: Edge Computing and the Internet of Things
It is an attempt at creating a smart routing system for garbage trucks using a distributed network of IOT-devices connected to garbage cans. Each garbage node (gnode) communicates its relative weight to an edge node over the COAP protocol, which then computes an optimal cycle for a garbage truck to drive through before sending this path to this website over http.
